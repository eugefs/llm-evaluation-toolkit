\# OpenAI-Compatible Provider



This package contains the OpenAI-compatible provider implementation for

`llm\_evaluation\_toolkit`.



\## Status



Current milestone: \*\*9.1 – Provider Skeleton\*\*



Implemented:



\- Configuration model

\- Exception hierarchy

\- Request/response models

\- Provider placeholder

\- Client placeholder



Not yet implemented:



\- OpenAI SDK integration

\- Authentication

\- Chat completions

\- Streaming

\- Retry logic

\- Error translation



\## Planned Compatibility



The provider is designed to work with any API implementing the OpenAI Chat Completions API, including:



\- OpenAI

\- Ollama

\- LM Studio

\- OpenRouter

\- LocalAI

\- vLLM

\- Text Generation Inference (OpenAI-compatible mode)



Future milestones will add networking and SDK support.
